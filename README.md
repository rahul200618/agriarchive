# AgriArchives

AgriArchives is an international monthly agriculture e-magazine dedicated to publishing high-quality research and articles in the field of agriculture.

## Project Overview

This project is a React-based web application built with Vite, TypeScript, and Tailwind CSS. It features a modern, responsive design and integrates with Supabase for backend services.

### Key Features

-   **Responsive Design**: Optimized for both desktop and mobile devices.
-   **Article Management**: Browse and search current and archived issues.
-   **Editorial Board**: Profiles of editorial board members.
-   **Submission Guidelines**: Information for authors regarding submission and fees.
-   **Search Functionality**: Real-time search for articles, issues, and static pages.

## Getting Started

To run the project locally:

1.  **Clone the repository**
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Start the development server**:
    ```bash
    npm run dev
    ```

## Technologies

-   **Frontend**: React, TypeScript, Vite
-   **Styling**: Tailwind CSS, shadcn/ui
-   **Backend**: Supabase (Database & Storage)
-   **Icons**: Lucide React
-   **Routing**: React Router DOM
